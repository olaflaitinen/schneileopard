# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog and this project adheres to Semantic Versioning.

## Unreleased

### Added

- Initial stable API for core domain types (GeneId, TranscriptId, ProteinId, SampleId, CohortId)
- Omics data structures (ExpressionMatrix, FeatureMatrix, SampleMetadata, Cohort)
- Validation framework for domain models
- Graph types for pathways and interaction networks
- Feature ranking and scoring infrastructure
- Sample stratification and clustering interfaces
- Explainability types for AI workflows
- CSV and TSV parsing for omics data
- Comprehensive test suite with unit, property-based, and integration tests
- Scaladoc for all public APIs
- Examples application demonstrating core workflows
- Benchmark scaffolding using JMH
